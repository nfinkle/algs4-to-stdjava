HashMap (Javadoc API) works almost identically to SeparateChainingHashST (Javadoc API).

The main difference between SeparateChainingHashST and HashMap is the difference in the names of the functions. The implementation of HashMap also uses a TreeMap (the equivalent of a Left-Leaning Red-Black BSTs) instead of a linked lists for each bucket in the hashtable that get too large to improve lookup time.
