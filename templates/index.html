Description

Below are a list of modules that correspond to classes in the standard Java libraries. Each one has a (roughly) equivalent that is used in algs4, the Java libraries used in Princeton's COS 226.

In each module, there are:

  • a short description of the differences between the standard Java class and the equivalent algs4 class,
  • an abridged API comparison, detailing most of the common and significant methods in each class, and
  • a progamming practice problem in which one is asked to translate a program written using the algs4 libraries into a program using only standard Java libraries.
The progress bars on this page keep track of your progress in completing the modules. Please complete the System.out and Scanner modules before the rest of the modules, as the rest of the modules require knowledge of those classes.
