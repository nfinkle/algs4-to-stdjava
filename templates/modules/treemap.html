TreeMap (Javadoc API) works almost identically to RedBlackBST (Javadoc API).

The main difference between RedBlackBST and TreeMap is that TreeMap can be used with Comparable keys or with an explicit Comparator, while RedBlackBST can only be used with Comparable keys. A Comparator for TreeMap must be of type Comparator<? super Key>, which means that it must be able to compare two objects of type Key and of any type from which Key inherits. For example a Comparator<? super Queue<T>> must accept arguments of the type Iterable<T> and Object because Queue<T> inherits from those interfaces.
