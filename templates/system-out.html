StdOut (Javadoc API) works almost identically to System.out (Javadoc API). One can print to stderr the exact same way as printing to stdout, except that you use System.err instead of System.out.

The only difference between StdOut and System.out/System.err is that StdOut flushes all output immediately, while System.out/System.err buffers the output until a newline character is added.
