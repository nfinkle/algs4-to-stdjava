Motivation

  • Princeton students do not learn how to code in Java using standard libraries, which are used at industry level.
  • Therefore, students have difficulty in interviews and generally programming in Java.

Goal

  • The goal of this project is to teach Princeton Computer Science students the core standard Java libraries that are replaced by algs4, which is the set of Java libraries used in COS 226.

Why was this project created?

  • Main motivation: Impress Prof. Robert Fish.
  • The project leader personally had difficulty transitioning from using algs4 to standard Java libraries and wished there was a tool to help assist him in adapting to the standard Java libraries for a Princeton student. So, he set out to make algs4 to StdJava.
